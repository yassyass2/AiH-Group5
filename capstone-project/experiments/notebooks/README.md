# Notebooks

These notebooks are kept for experiment traceability and team review. They may use Google Colab setup cells, but the assumptions should be visible in the first cells.

Notebook outputs are cleared before committing. Selected results are stored next to the notebook under `../reports/`.

Older exploratory notebooks can live here when they directly support the APTOS capstone work. General course assignments should stay under the repository-level `docs/coursework` folder.
